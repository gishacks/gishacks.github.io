# تحلیل شبکه
در قسمت catalog در قسمت databases پس از اضافه کردن لایه شبکه ی معبر ما میخواهیم این لایه ی وکتوری را به لایه ی شبکه ای تغییر دهیم. از قسمتnew یکfeatufe data set جدید ایجاد می کنیم و اطلاعات را برای تهران  درست می کنیم. (39N) و بعد از قسمت import لایه را به کاتالوگ انتقال می دهیم.
در قدم دوم یک  network dataset میسازیم و لایه ی مورد استفاده را از لایه انتقالی ما به کاتالوگ انتخاب می کنیم و سپس runمی کنیم.
# service area
از سربرگ analysis گزینه ی network analysisرا انتخاب کرده و از موارد در دسترس servise areaرا انتخاب می کنیم.
برای بخش تنظیمات آن ابتدا در مورد fascility لایه ی بیمارستان را تایید کرده و پوشش را بر ابعاد 1000و2000و5000 قرار می دهیم. سپس در قسمت travel attributes این ستاپ اضافه شده را ثبت می کنیم.
# point density
در کنار هر مکان نقطه ای یک دایره میزند و تراکم را در آن میسنجد (با توجه به اطلاعات داده شده به ابزار)
این ابزار برای مقایسه های جزئی بکار می رود.(جزئیات دوایر مقیاسی دیده می شود.)
# kernel density
در صورت نیاز به محاسبات کلی میتوانیم از این مورد استفاده کنیم همان کار های point را انجام می دهد ولی نتیجه را به صورت داده ی کلی و بر اساس تمامی دوایر خروجی میگیرد.
# feature to point
برای قسمت هایی که میخواهیم اطلاعات مورد استفاده را بیشتر کنیم و یک فایل داریم که به صورت نقشه ای وجود دارد می توانیم از این ابزار  استفاده کنیم زیرا برای point density نیاز به لایه های نقطه ای داریم.معمولا تراکم به صورت 1000 برای هر کیلومتر مربع استفاده می شود.
