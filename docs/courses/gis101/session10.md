#استفاده از ابزار join
برای مرتبط کردن جدول Attribute یک لایه یا جدول به لایه‌ی دیگر از ابزارjoin استفاده می‌کنیم.

برای اجرای ابزار join  ما نیاز به عامل مشترک داریم؛ این عوامل مشترک می‌توانند به صورت فیلد مشترک در دو جدول، دو مکان مشترک در دو نقشه، یا دو لایه باشند که از لحاظ جغرافیایی بر هم تطابق دارند.

استفاده از ابزار join با توجه به جدول attribute:

در این قسمت از جدول pivot که پیش‌تر ساخته‌ایم، استفاده می‌کنیم.

نکته: قبل از وارد کردن جدول به GIS  باید آن را به جدول خام تبدیل کنیم.

تبدیل جدول pivot به جدول خام:

1-     ابتدا تمام جدول را کپی می‌کنیم

2-     یک فایل اکسل جدید باز می‌کنیم

3-    در قسمت A1 راست کلیک کرده و سپس Paste

نکته: جدول حتما  باید از A1 شروع شود. در این قسمت  باید داده‌های سطر اول پاک شود زیرا نام ستون‌ها روی آن‌ها ثبت شده است. در سطر اول روی شماره 1 رایت کلیک کرده  و گزینه delete را می‌‌زنیم؛ سپس اسم ستون‌هایمان را عوض می‌کنیم.

نام ستون‌ها از چپ به راست: 
aid-rest-harim

پس از انجام این مراحل باید جدول را به gis ببریم:
save as/ format: Excel 97-2003/ 
سپس gis را باز می‌کنیم و لایه محلات را فقط فعال نگه‌می‌داریم؛ جدول اکسل را از add data باز می‌کنیم و سپس: 
right click/ join and relates/ add join/ layer field/ mah_id/ join table: export table/ table field: aid/ validate

اما نکته اینجاست که join  دائمی نیست و برای دائمی کردن اطلاعاتی که پس از انتخاب join  برای‌مان مهم واقع شده‌اند، لازم است که در حین فعال بودن join  یک فیلد جدید را با استفاده از add field  بسازیم و سپس با استفاده از ابزار calculate field داده‌های لازم را با انجام اعمال ریاضی در فیلد جدید ثبت کنیم. همچنین مشخصات فیلد جدید: 
name: harim/ data type: double/ format: percentage/ rounding: 2

پس از ساخت فیلد جدید و ثبت داده‌ها با استفاده از calculate field، حالا می‌توانیم join را حذف کنیم:
انتخاب لایه مورد نظر/ کلیک راست/ joins and relates/ سپس انتخاب remove all joins.

همچنین برای تعیین محله‌های آسیب‌پذیر می‌توانیم با استفاده از symbology و حالت graduated colors با انتخاب رنگ، شدت آسیب‌پذیری را تعیین کنیم.
علاوه بر آن برای اینکه نام خیابان‌ها و کوچه‌ها و... را بتوانیم ببینیم باید از این راه برویم:
feature layer/ layer blend/ multiply
